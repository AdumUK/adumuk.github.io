Completing the project involved:

  • Converting a large majority of the code base from VB.Net to C#

  • Implementing client approved designs, partially moved legacy styling into Sass and built the new design logic on top as some aspects of the site remained the same e.g. admin controlled areas.
  • Business logic changed from single product purchase to multiple types of subscriptions, required migration of legacy clients to new subscription system along with tools to enable staff to manage/create subscriptions.
  • Stripe payment processor incorporated into the site and updates to existing PayPal IPN
  • Fully incorporating Bootstrap 3 to assist with making the site mobile compatible, along with custom responsive designs.
